# PrivateDevelop
個人開発に使うルートリポジトリ
ブランチ名はプロジェクト名
PRを作成することでフォルダを増やしていく

## Usage
初期mainには何にもなくて，PRをmergeすることでフォルダをを増やしていく
```
main
|_ projectAのPRが作るprojectA dir
|_ projectBのPRが作るprojectB dir
|_ projectCのPRが作るprojectC dir
...
```
